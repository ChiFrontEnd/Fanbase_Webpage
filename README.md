# 🎶 Cortis Fanbase Website

A multi‑page **official artist hub** built to celebrate **CORTIS**, showcasing their journey, discography, achievements, and fan community.

---

## 🌐 Project Overview
The Cortis Fanbase Website is a **responsive, multimedia fan hub** featuring:
- 🎥 Hero section with autoplay video background + parallax effect
- 📊 About & Biography with stats and milestones
- 👥 Members section with interactive modals
- 🎼 Discography carousel highlighting albums
- 📺 Embedded YouTube music video
- 🖼️ Gallery with lightbox viewer
- 📰 Press & Media coverage cards
- 🏆 Awards timeline
- 📅 Schedule table for events
- 📰 News updates
- 🌍 Language switcher (English, Korean, Japanese)
- 🔗 Social media links
- 📌 Footer with branding

---

## 🛠️ Tech Stack
- **HTML5** for structure  
- **CSS3** + custom styling (`Style.css`)  
- **Bootstrap 5.3.3** for responsive layout  
- **Bootstrap Icons** for social/media icons  
- **JavaScript (`Script.js`)** for interactivity:
  - Loader animation  
  - Sticky navbar  
  - Smooth scroll  
  - Gallery lightbox  
  - Scroll reveal animations  
  - Language switcher  
  - Counter animation for stats  
  - Active nav link highlight  
  - Hero parallax effect  

---

## 🎨 Design & Styling Highlights
- **Dark theme** with neon accent (`#ccff00`)  
- **Glassmorphism effects** on navbar and overlays  
- **Hover animations** on cards, gallery, and stats  
- **Responsive grid layouts** for members, gallery, and socials  
- **Timeline design** for awards section  
- **Lightbox viewer** for gallery images  
- **Custom scroll animations** (`fade-up`, `fade-in`)
